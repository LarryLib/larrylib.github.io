# 内存分类
- Leaded Memory
- Abandoned Memory
- Cached Memory

# 常见的内存泄漏
- 僵尸对象和野指针
    - 僵尸对象：内存被回收的对象
    - 野指针：指向僵尸对象的指针，向野指针发送消息会导致崩溃。
- 循环引用
- 循环中对象占用内存大
- 无限循环
- 系统内存警告
