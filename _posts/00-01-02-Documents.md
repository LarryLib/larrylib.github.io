# 批量操作命令(注：文件名不可以有空格)
打印当前路径下所有的文件夹
```ruby
for dir in $(ls -d */);do
echo $dir
done
```

打印当前路径下所有的文件
```ruby
for file in $(ls ./);do
echo $file
done
```

```ruby
for file in $(ls ./);do
echo "- [${file}](https://wuxuange.com/Document/OpenGL ES 2.0/${file})"
done
```
